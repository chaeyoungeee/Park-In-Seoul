# 🌳 Park In Seoul

##  💡 소개
**파크 인 서울** 은 서울시 실시간 도시데이터 API에서 제공하는 공원 29곳의 `인구 혼잡도`, `날씨 및 미세먼지`, `행사` 등의 실시간 데이터를 제공하여 사용자의 공원 이용 편리성을 높여줍니다!

## 💡 기술 스택

## 💡 메인 기능 및 서비스 화면
- 29곳의 공원 실시간 정보를 제공합니다.
  - 공원의 혼잡도, 날씨 정보를 제공합니다.
  - 혼잡도 높은 순/낮은 순으로 정렬하여 확인할 수 있습니다.
